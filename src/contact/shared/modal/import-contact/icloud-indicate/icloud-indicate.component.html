How to import iCloud Contacts into WTH!Contact

Please follow these steps to import iCloud Contacts into WTH!Contact

  1. Open Contacts or Address Book

  2. Chosse File > Export > Export vCard option or simply drap and drop All Contacts from the Group list to your desktop. You can select one or more specific contacts if you'd rather not export the whole list.
    If you do not see All Contacts, select View -> Show Groups from the menu

  3. Open WTH!Contact

  4. Select Import > iCloud Contact from the menu

  5. Select the "All Contacts.vcf" from the desktop (created in Step 2) and press Open. The vfc file will be imported into your WTH!Contact
